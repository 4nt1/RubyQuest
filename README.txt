RubyQuest Version 0.1
Nécessite une version de Ruby 1.9.*
 
 ************* Procédure d'installation Windows *******************

 1) Télécharger Ruby à cette adresse cdn.rubyinstaller.org/archives/1.9.3-p327/rubyinstaller-1.9.3-p327.exe
 2) Installer le fichier : 	à l'installation cocher "Associer les fichier .rb ...."
 							vous pouvez aussi cocher le reste
 3) Retourner dans le dossier RubyQuest, lancer le jeu en cliquant sur le fichier game.rb
 	Il est recommandé d'agrandir la taille de la console : Clic droit sur le terminal, propriétés
 
		 ************************************************************************************************************
		 * SI VOUS N'ARRIVEZ PAS A LE LANCER DE CETTE MANIERE                                    					*
		 * (vous n'avez probablement pas coché la case "associer les fichiers .rb...")         						*
		 * - Déplacez le dossier RubyQuest sur D: (par exemple)                        						        *
		 * - Ouvrez la console Ruby (menu démarrer, Tous les programmes, Ruby, Start command prompt with Ruby)      *
		 * - Dans la console tapez :                                                    						    *
		 * 	D:                                                                            						    *
		 * 	cd RubyQuest                                                                  					        *
		 * 	ruby game.rb                                                                  				       		*
		 ************************************************************************************************************


 ************* Procédure d'installation Ubuntu 12.04 (et dérivés, Mint...) *******************
 1) Ouvrez un terminal et tapez 
 	sudo apt-get install ruby1.9.1-full

 2) Placez le dossier RubyQuest dans un dossier (par exemple Documents)

 3) Ouvrez un terminal et déplacez vous jusqu'au dossier où est RubyQuest avec cette commande :
 	cd /home/monNomdUtilsateur/Documents/RubyQuest
 	en remplaçant monNomdUtilsateur par votre nom d'utilisateur (sans majuscule)

 4) Executez le fichier de démarrage en tapant
 	ruby game.rb
